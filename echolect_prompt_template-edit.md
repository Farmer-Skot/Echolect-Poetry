# ECHOLECT CONSTRUCTION PROTOCOL
## A Prompt Template for Any LLM

---

## WHAT IS ECHOLECT?

A poem in which there are multiple, often open-ended, meanings. 
There is at least three levels of embedded meaning in an Echolect poem

**Similar to a concrete poem in which:**

"A poem of a bell, shaped like a bell".

**However, in an echolect poem it becomes:**

"A poem of a bell, shaped like na bell written every phonetic token ("tokenette") is memetically related to a set of constraints about a bell."

1. Phonosthetically chosen to match a specified set of English words, usually lyrics. ("ghost-english"). But might be nonesense words that sound phonesthetically interesting, e.g. "lalala" and "ooooh" or jazz scat in singing.

2. Each tokenette has meaning related to a theme. (Vibe story)

3. The entire poem is aesthetically non-standard script. Written in relative fonts, scripts, with visual asthetics in mind. (Concrete poem)

4. Has at least two or more constraints: e.g. I. Ghost_lyrics("...") II. Seed_phrase("water, light, stream, cut") AND/OR III. Vibe_story:(i.e. Only uses words related to the sound or activity of ringing bells) IV. Vibe_constraint("Only use words that existed prior to the 3rd century.), etc. Many options are possible.

**The result:**

Speak the poem aloud and hear English. 
Read the meanings and find a hidden second poem. 
View the poem and find a hidden third poem. And so on.
Allow for open-ended artistic recombinations of words with something like a "soft high temperature" approach.
Allow glossolalia and adjust for ghost lyrics.
A beautiful blend of 
1. Embedded meaning
2. Phonesthitic aesthetics 
3. Font-level visual intruige is the primary goal of an echolect poem.

---

## THE AIM

1. Tokenette's that are real words, or word stems from a dictionary language (note: from any language real or invented prodvided it has documentation of usage, e.g. Klingon, Phizora, Gnostic Zalgo, etc.)** Compounds (e.g., हिम-høj) allowed if EACH part is independently real. (This is where it gets fun.)
OR
2. Every tokenette is charged with meaning and phonthestics to match a vibe story.
OR
3. Both made up of real words, and phonosthetic glossolalia, or hybrids.
4. Other constraints allowable.
5. Pronunciation should either match the intended ghost enligh lyrics, or, be how **native speakers actually say it**. Not necessarily both. Default and adapt based on th other constraints.
6. Poems that are visually beautiful, accounting for the user need to adjust any ASCII to local enbvironments. Err in using spaces instead of tabs when creating ASCII concrete poetic instantiations as it translates better with copy and paste.

---

### STEP 1: DECOMPOSE
Break ghost-English into phonetic targets:
```
"Open the door" → "oh-pen" + "da" + "dor"
```

### STEP 2: HUNT
For each phonetic target, search ALL language families for real words that:
- SOUND like the target
- MEAN something in the vibe constraint
- Allow unexpected word choices to emerge.

### STEP 3: ASSEMBLE
Write three forms:
- **Ghost-English**: What the listener hears
- **Echolect**: The concrete poem
- **Vibe_Story**: Token meanings read as their own poem
- **Unicode**: Use the original language unicode for aesthetics

### STEP 4: VERIFY
- Confirm each word is real (dictionary check)
- Confirm pronunciation (IPA or close approximation)
- Count: aim for 4+ languages, 3+ scripts

```

### Example 1: 
**Title** 
"Gate Less Gate" — 100% Verification
**Ghost_English:**
"Open the door / The door is open / Walk through the door / I go, I go, I go"
**Vibe_Constraint:** 
Only use tokenettes memetically related to "threshold, passage, crossing".
***Concrete_Constraint:**
 "Poem shaped like a gateway"
**Concrete_Form:**
```
Öppen ده dør / ده dør іс öppen / wâc þrú ده dør / 愛 고 愛 고 愛 고

```
| Token | Lang | Means | Ghost |
|-------|------|-------|-------|
| Öppen | Swedish | open | open |
| ده | Pashto | this | the |
| dør | Norwegian | door | door |
| іс | Kazakh | deed | is |
| wâc | Welsh | a stroll | walk |
| þrú | Old Norse | through | through |
| 愛 | Chinese | love | I |
| 고 | Korean | high/to go | go |

**Vibe_Story:** *Openness — this door. This door is deed. A stroll through this door. Love rises, love rises, love rises.*
**Key_find:** 愛 (love) = "I". The one who passes through the gate is love itself.
